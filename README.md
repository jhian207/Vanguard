# Vanguard 🛡️
**The Front Line of Server Integrity.**

Vanguard is a high-performance moderation engine designed to maintain order, enforce rules, and protect your community from bad actors. It’s not just a bot; it’s your server’s digital peacekeeping force.

---

### 🚀 Key Features
* **Automated Moderation:** Real-time filtering for banned words, spam detection, and mass-mention protection.
* **Logging System:** Detailed audit logs for deleted messages, edited content, and member updates.
* **Advanced Punishment Logic:** Seamlessly execute kicks, bans, soft-bans, and timed mutes.
* **Verification Gates:** Prevent raid attacks by requiring new users to pass specific security checks before gaining access.

### 🛠️ Core Commands
| Command | Description | Usage |
| :--- | :--- | :--- |
| `/ban` | Permanently remove a user from the server. | `/ban @user [reason]` |
| `/mute` | Temporarily restrict a user's ability to speak. | `/mute @user [duration]` |
| `/purge` | Bulk delete messages from a specific channel. | `/purge [amount]` |
| `/slowmode` | Adjust the rate of chat in a specific channel. | `/slowmode [seconds]` |

### ⚙️ Installation
1. Invite Vanguard using the [Invite Link](#).
2. Ensure the `Vanguard` role is at the top of your hierarchy.
3. Run `/setup-mod` to configure your logging and alert channels.
